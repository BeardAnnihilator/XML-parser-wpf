## XML Parser in WPF
--
#### Schema

```
<acousticmaterial>
    <mtllib>
        <material name="ABC" absorption="0.41 0.34" scattering="0.67 0 0.24"/>
    </mtllib>
</acousticmaterial>
```

#### Requirements

1. Display all materials (_10,000+_) in XML using WPF Gui.
2. Allow user to search _materials_ by **name**
